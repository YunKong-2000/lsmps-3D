# PPE行修改法
## 问题背景
实现显式的自由面压力边界条件，直接对PPE中自由面粒子对应的行进行修改。使得对自由面粒子i有$p_i=0$
## 具体实现
首先照常按照原有方法计算PPE系数矩阵，其中在计算自由面粒子的系数式直接令对角线元素为常数$c$，右边项为0，其余非对角线元素为0。其中常数$c$的大小最好接近非自由面粒子对应的对角线元素的值，这样可以保证系数矩阵的条件数不会过大。暂定$c=\frac{1}{\Delta x^2 \rho}$
最终求解线性方程组$Ap=b$。
## 存在的问题
常数$c$是需要额外计算的。另外对于自由面粒子来说，这样的处理相当于忽略了其邻域中的其他粒子对其的影响，但是在其邻域粒子所对应的行中，它的影响并未忽略，这就导致了单向的耦合关系。
