# moment矩阵计算和管理
## 目的
lsmps方法中的算子离散需要预先计算moment矩阵$M^{-1}$。此模块将完成所有的算子对应的moment矩阵的计算并将存储这些矩阵以供后续模块调用
## 输入
流体粒子及壁面粒子的位置，壁面法向。
## 输出
一系列显式的moment逆矩阵 $M^{-1}$，通过只读 device view 提供给其他模块。

## 具体实现
模块将计算并存储以下 moment 逆矩阵：
1、速度对应算子的 $M^{-1}$，Type-A 格式，包含壁面 Dirichlet 样本贡献。
2、压力对应算子的 $M^{-1}$，Type-A 格式，包含壁面第二类边界条件矩阵贡献。
3、压力近自由面或后续修正可能使用的 $M^{-1}$，Type-B 格式。
4、仅流体邻域的 Type-A $M^{-1}$，作为后续扩展备用。

moment_matrix 模块不再提供具体算子离散方法。provision、PPE、correction 等模块在接收对应的 moment 逆矩阵后，自行构建 RHS、乘以 $M^{-1}$ 并提取所需梯度、散度或拉普拉斯系数。
